# Interface Responsiva Mobile-First

Use esta skill em qualquer tarefa de interface do Bolso em Dia.

## Regras principais

- Pensar toda interface primeiro para celular.
- Garantir que o sistema também funcione bem em telas de computador.
- Evitar layouts largos demais no celular.
- Usar cards, espaçamentos e botões confortáveis para toque.
- Criar formulários simples, com campos bem visíveis.
- Manter listas legíveis em telas pequenas.
- Evitar tabelas grandes no mobile; preferir cards ou listas responsivas.
- Fazer filtros funcionarem bem no celular.
- Manter menus e navegação simples.
- Adaptar dashboard para celular e desktop.
- Verificar responsividade antes de considerar a tarefa concluída.
- Considerar estados de vazio, erro, carregamento e sucesso ao implementar interface.

## Checklist de conclusão

- Verificar ao menos uma viewport de celular.
- Verificar uma viewport de desktop quando houver layout em colunas.
- Confirmar que textos não sobrepõem outros elementos.
- Confirmar que botões e campos são fáceis de tocar.
- Confirmar que listas e cards continuam legíveis.
