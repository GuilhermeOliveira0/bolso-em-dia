# Dashboard Financeiro

Use esta skill ao criar, alterar ou revisar telas e dados de dashboard financeiro.

## Princípios

- Manter o dashboard simples, visual, mobile-first e responsivo para celular e computador.
- Priorizar clareza para o usuário entender onde está gastando mais.
- Evitar excesso de informação.
- Mostrar primeiro os dados mais importantes.
- Mostrar apenas dados do usuário autenticado.
- Nunca agregar gastos de usuários diferentes.

## Filtros

O dashboard deve permitir filtros por:

- mês;
- ano;
- categoria;
- tipo do gasto;
- forma de pagamento;
- recebedor;
- valor mínimo e máximo.

## Cards principais

Incluir cards para:

- gasto total do mês;
- total necessário;
- total lazer;
- total supérfluo;
- economia possível.

## Gráficos

Incluir gráficos para:

- gastos por categoria;
- gastos por tipo;
- evolução mensal.

## Rankings

Incluir rankings para:

- maiores recebedores;
- maiores gastos;
- categorias com maior aumento.

## Cuidados de experiência

- Evitar telas poluídas.
- Usar textos curtos e diretos.
- Garantir boa leitura em celular.
- Garantir boa leitura em desktop sem criar telas largas demais no mobile.
- Permitir que filtros mudem cards, gráficos e rankings de forma consistente.
- Validar responsividade antes de considerar a dashboard concluída.
