# User Stories

## Cadastro manual

Como usuário,
quero cadastrar um gasto manualmente,
para manter meu controle financeiro atualizado mesmo sem comprovante.

Como usuário,
quero informar valor, data, categoria e tipo do gasto rapidamente,
para registrar o gasto pelo celular sem perder tempo.

## Comprovante Pix

Como usuário,
quero enviar um comprovante Pix,
para o sistema preencher dados do gasto automaticamente.

Como usuário,
quero enviar imagem ou PDF do comprovante,
para usar o arquivo que eu já tenho no celular.

## Revisão dos dados extraídos

Como usuário,
quero revisar os dados extraídos do comprovante,
para corrigir qualquer erro antes de salvar.

Como usuário,
quero ver campos com baixa confiança destacados,
para saber o que preciso conferir com mais atenção.

## Categoria sugerida

Como usuário,
quero receber uma categoria sugerida,
para classificar o gasto mais rápido.

Como usuário,
quero corrigir a categoria sugerida,
para manter meus registros fiéis ao meu jeito de organizar gastos.

Como usuário,
quero que o sistema aprenda com a categoria corrigida,
para receber sugestões melhores no futuro.

## Tipo sugerido

Como usuário,
quero receber um tipo de gasto sugerido,
para entender se o gasto foi necessário, lazer, supérfluo ou outro tipo.

Como usuário,
quero corrigir o tipo sugerido,
para ajustar a análise financeira à minha realidade.

Como usuário,
quero que o sistema aprenda com o tipo corrigido,
para melhorar futuras classificações.

## Dashboard

Como usuário,
quero visualizar uma dashboard,
para entender rapidamente como estão meus gastos no mês.

Como usuário,
quero filtrar gastos,
para analisar um período, categoria, tipo, forma de pagamento, recebedor ou faixa de valor.

Como usuário,
quero ver ranking de maiores gastos,
para identificar os gastos que mais impactaram meu orçamento.

Como usuário,
quero acompanhar gastos supérfluos,
para encontrar oportunidades de reduzir despesas.

Como usuário,
quero visualizar evolução mensal,
para comparar meu comportamento ao longo do tempo.

## Limite mensal e economia

Como usuário,
quero definir um limite mensal,
para acompanhar se estou gastando dentro do planejado.

Como usuário,
quero entender minha economia possível,
para saber onde posso reduzir gastos sem depender de análise manual.

## Assumptions

- As histórias priorizam usuário individual.
- Histórias de administração, cobrança e compartilhamento ficam fora do MVP.
